# Majestic Motors Gallery

A comprehensive **Showroom Management System** designed to streamline vehicle inventory, customer management, employee tracking, and service operations for an automotive dealership.

## Project Overview

- **Type:** Console-Based Application
- **Language:** C++
- **Subject:** Programming Fundamentals (1st Semester)
- **Purpose:** Demonstrate fundamental programming concepts

## Features

### Core Functionality
- **Vehicle Management:** Add, update, delete, and view vehicle inventory
- **Customer Management:** Maintain customer records and purchase history
- **Employee Management:** Track employee information and roles
- **Manager Portal:** Administrative controls and system oversight
- **Test Drive Management:** Schedule and manage test drives
- **Vehicle Service:** Track maintenance and service records
- **Comments System:** Customer feedback and comments tracking

### User Roles
- **Admin:** Full system access with management privileges
- **Managers:** Operational oversight and reporting
- **Employees:** Daily operational tasks
- **Customers:** View vehicles and schedule test drives

## Data Management

The system uses text-based data files for persistent storage:

| File | Purpose |
|------|---------|
| `Vehicles_Data.txt` | Complete vehicle inventory |
| `Customers_Data.txt` | Customer profiles and details |
| `Employees_Data.txt` | Staff information and roles |
| `Managers_Data.txt` | Management accounts |
| `Products_Data.txt` | Additional products/services |
| `TestDrives_Data.txt` | Test drive bookings |
| `VehicleService_Data.txt` | Maintenance and repairs |
| `Comments_Data.txt` | Customer reviews and feedback |
